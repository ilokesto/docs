# ilokesto docs

Official bilingual documentation for the ilokesto package family.

## Routes

- `/en/state`
- `/ko/state`
- `/en/store`
- `/ko/store`
- `/en/utilinent`
- `/ko/utilinent`

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Stack

- Next.js 16
- Fumadocs
- MDX content in `content/docs`

## Maintainer Guide

- `DOCS_ARCHITECTURE.md` — the single source of truth for official docs structure
