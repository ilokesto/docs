# Store API

## `new Store<T>(initialState: T)`
초기 상태와 함께 Store 인스턴스를 생성합니다.

## `store.getState(): Readonly<T>`
현재 상태의 스냅샷을 반환합니다.

## `store.getInitialState(): Readonly<T>`
생성 시 제공된 초기 상태를 반환합니다.

## `store.setState(nextState: T | ((prevState: Readonly<T>) => T)): void`
상태를 새 값으로 교체하거나 이전 상태를 기반으로 다음 상태를 계산합니다. `Object.is(prevState, nextState)`가 `true`인 경우 구독자에게 알리지 않습니다.

## `store.subscribe(listener: () => void): () => void`
상태 변경 시 실행될 리스너를 등록합니다. 구독 취소 함수를 반환합니다.
