# @ilokesto/state

`@ilokesto/state`는 `@ilokesto/store` 위에 얹는 React용 상태 관리 패키지입니다. `create()` API, 셀렉터 기반 구독, 리듀서 모드, 미들웨어 유틸리티를 통해 React 컴포넌트에서 store를 다루기 쉽게 만들어 줍니다.

## 설치

```bash
pnpm add @ilokesto/state react
```

`adaptor()`를 사용할 예정이라면 `immer`도 함께 설치하세요.

```bash
pnpm add immer
```
