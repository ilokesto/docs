# @ilokesto/state

`@ilokesto/state` is a focused state management library for React. It aims to provide the perfect balance between simplicity and scalability by focusing on **store-hook unity**.

## Core Philosophy

- **No Providers**: No need to wrap your application in complex Context Providers.
- **Predictable**: State is external but deeply integrated with React's rendering through `useSyncExternalStore`.
- **Composable**: Built-in support for middleware like logging, persistence, and validation.
- **Performant**: Optimized re-renders through a powerful selector system.

## Installation

```package-install
@ilokesto/state
```

## Quick Look

```tsx
import { create } from '@ilokesto/state';

const useCounter = create({ count: 0 });

function Counter() {
  const [count, setState] = useCounter((state) => state.count);

  return (
    <button onClick={() => setState((prev) => ({ ...prev, count: prev.count + 1 }))}>
      {count}
    </button>
  );
}
```
