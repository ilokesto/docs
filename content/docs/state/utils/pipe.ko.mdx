# `pipe`

`pipe`는 plain initial state를 기준으로 여러 미들웨어를 조합하고, 최종적으로 초기화된 `Store` 인스턴스를 반환합니다.

```ts
import { create } from '@ilokesto/state';
import { pipe } from '@ilokesto/state/utils';
import { logger, devtools } from '@ilokesto/state/middleware';

const store = pipe(
  { count: 0 },
  logger(),
  devtools('MyStore')
);

const useStore = create(store);
```

## 동작 방식

- plain initial state에서 시작합니다.
- vanilla `Store`를 생성합니다.
- 미들웨어를 왼쪽에서 오른쪽으로 적용합니다.
- 결과 `Store`를 반환하고, 이를 `create(store)`에 전달할 수 있습니다.

## 언제 쓰면 좋은가

여러 미들웨어를 수동으로 중첩하지 않고, 더 읽기 좋은 형태로 조합하고 싶을 때 `pipe`를 사용하는 것이 좋습니다.
