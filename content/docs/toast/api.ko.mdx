# API 레퍼런스

`@ilokesto/toast`는 named value만 export합니다.

## 값 export

### `toast`

facade 표면은 다음과 같습니다.

```ts
toast(message, options?)
toast.success(message, options?)
toast.error(message, options?)
toast.loading(message, options?)
toast.custom(message, options?)
toast.promise(promiseOrFactory, messages, options?)
toast.dismiss(id?, toasterId?)
toast.remove(id?, toasterId?)
```

중요한 동작:

- `toast.loading()`은 infinite duration을 강제하고
- `toast.promise()`는 `Promise<T>`와 `() => Promise<T>` 둘 다 받으며
- `dismiss()`는 soft close이고
- `remove()`는 즉시 삭제입니다

### `Toaster`

하나의 runtime을 렌더링하는 메인 component입니다.

자세한 prop은 [Toaster](./toaster)에서 다룹니다.

### `ToastBar`

기본 row renderer primitive입니다.

자세한 내용은 [ToastBar](./toast-bar)에서 다룹니다.

### `ToastIcon`

기본 row에서 사용하는 built-in icon renderer입니다.

### `useToaster()`

headless hook이며 다음을 반환합니다.

```ts
{
  toasts,
  handlers: {
    updateHeight,
    startPause,
    endPause,
    calculateOffset,
  }
}
```

### `useToastItems(runtime)`

runtime의 visible toast item에 대한 low-level external-store subscription입니다.

### `createToastRuntime(toasterId)`

toast runtime 인스턴스 하나를 생성합니다.
