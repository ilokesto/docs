# 유틸리티 (Utils)

`@ilokesto/state/utils` 엔트리포인트는 두 가지 주요 헬퍼인 `pipe`와 `adaptor`를 제공합니다. 이들은 스토어 설정과 상태 관리를 더 인체공학적으로 만들기 위해 설계되었습니다.

## `pipe`

`pipe`는 초기 상태를 기반으로 여러 미들웨어를 조합하는 유틸리티입니다. 미들웨어를 왼쪽에서 오른쪽(위에서 아래)으로 적용하고, 완전히 초기화된 `Store` 인스턴스를 반환합니다.

```ts
import { create } from '@ilokesto/state';
import { pipe } from '@ilokesto/state/utils';
import { logger, devtools } from '@ilokesto/state/middleware';

const store = pipe(
  { count: 0 },
  logger(),
  devtools('MyStore')
);

const useStore = create(store);
```

여러 미들웨어를 사용하는 복잡한 스토어를 설정할 때 `pipe`를 사용하는 것이 좋습니다.

## `adaptor`

`adaptor`는 Immer의 `produce`를 얇게 래핑한 것입니다. draft 스타일의 상태 레시피를 작성하면, 그 결과를 순수 updater 함수로 돌려줍니다.

```ts
import { adaptor } from '@ilokesto/state/utils';

type State = { user: { profile: { name: string } } };

const renameProfile = adaptor<State>((draft) => {
  draft.user.profile.name = 'New Name';
});

useStore.writeOnly()(renameProfile);
```

### 설치

`adaptor`를 사용하려면 프로젝트에 `immer`가 설치되어 있어야 합니다.

```package-install
immer
```

## 관련 API

- [`create`](./create): React 통합을 위한 메인 진입점.
- [`미들웨어`](./middleware): 일반적인 작업을 위한 내장 미들웨어.
