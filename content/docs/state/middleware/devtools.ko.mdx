# DevTools

`devtools` 미들웨어는 `@ilokesto/state` 스토어를 Redux DevTools 확장 프로그램에 연결하여 고급 디버깅을 제공합니다.

```ts
import { devtools } from '@ilokesto/state/middleware';

const store = devtools('CounterStore')({ count: 0 });
```

## 주요 기능

- **상태 검사**: Redux DevTools 확장 프로그램에서 전체 상태 트리를 확인할 수 있습니다.
- **액션 라벨**: 액션 이름을 알 수 있을 때 업데이트는 `${name}:${actionName}` 형식으로 전송됩니다.
- **메시지 처리**: 확장 프로그램에서 보내는 `RESET`, `COMMIT`, `ROLLBACK` 명령을 처리합니다.

## 제약 사항

- **브라우저 전용**: `window.__REDUX_DEVTOOLS_EXTENSION__`에 등록된 확장 프로그램에 의존합니다.
- **운영 환경 비활성화**: `logger`와 마찬가지로 운영(production) 환경에서는 자동으로 비활성화됩니다.
- **액션 명명 패턴**: 업데이트 정보는 `${name}:${actionName}` 또는 `${name}:anonymous action` 형식으로 DevTools에 전송됩니다.
