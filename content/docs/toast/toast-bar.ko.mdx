# ToastBar

`ToastBar`는 custom row renderer를 주지 않았을 때 `Toaster`가 사용하는 기본 row primitive입니다.

## Props

- `toast`
- `position?`
- `style?`
- `children?`

## 기본 동작

`ToastBar`는 세 가지를 주로 담당합니다.

- `toast.status`와 `position`으로 enter/exit motion style 계산
- `ToastIcon`을 통한 기본 아이콘 렌더링
- `children`으로 override하지 않는 한 기본 메시지 레이아웃 렌더링

`toast.type === 'custom'`이면 `toast.message`를 그대로 렌더링합니다.

## custom row layout

`children`이 render function이면 다음을 받습니다.

```ts
{ icon, message }
```

```tsx
<ToastBar toast={toast}>
  {({ icon, message }) => (
    <>
      {icon}
      <strong>{message}</strong>
    </>
  )}
</ToastBar>
```

## 언제 쓰면 좋은가

- 기본 motion/container 동작은 유지하면서 내부 레이아웃만 바꾸고 싶다면 `ToastBar`
- row 전체를 바꾸고 싶다면 `Toaster`의 custom row 렌더링
