# Toast (토스트)

`@ilokesto/toast`는 `@ilokesto/overlay`를 사용하여 생명주기를 관리하는 프로바이더 범위(provider-scoped)의 토스트 런타임 및 퍼사드(facade)입니다. 간단한 프로미스 기반 API를 통해 임시 알림을 표시할 수 있습니다.

## 이 패키지는 무엇인가요?

React를 위한 가벼운 알림 라이브러리입니다. 프로바이더 범위 패턴을 따르므로 애플리케이션 내에 고유한 설정을 가진 여러 토스트 영역을 가질 수 있습니다.

## 제공하는 기능

- **간단한 퍼사드**: 어디에서나 `toast.success()`, `toast.error()` 등을 호출할 수 있습니다.
- **프로바이더 범위**: 각 `Toaster`는 독립적으로 작동합니다.
- **풍부한 기본값**: 아름다운 아이콘과 스타일이 기본으로 제공됩니다.
- **프로미스 지원**: 모든 프로미스에 대해 로딩, 성공, 오류 상태를 쉽게 처리할 수 있습니다.
- **헤드리스 코어**: `useToaster`를 사용하여 나만의 커스텀 토스트 UI를 빌드할 수 있습니다.

## 의도적으로 하지 않는 것

- **전역 싱글톤 없음**: 토스트는 `Toaster`로 범위가 제한됩니다. 적어도 하나의 `Toaster`를 마운트해야 합니다.
- **수동 생명주기 관리 불필요**: 토스트는 설정된 지속 시간(duration)에 따라 자동으로 사라지고 제거됩니다.

## 간단한 예제

```tsx
import { Toaster, toast } from '@ilokesto/toast';

function App() {
  return (
    <div>
      <Toaster />
      <button onClick={() => toast.success('저장되었습니다!')}>저장</button>
    </div>
  );
}
```
