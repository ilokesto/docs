# 로거

`logger` 미들웨어는 모든 상태 업데이트를 추적하여 브라우저 콘솔에 출력합니다. 이전 상태와 다음 상태를 보여주고, 가능하다면 로그 제목에 액션 이름도 함께 남깁니다.

```ts
import { logger } from '@ilokesto/state/middleware';

const store = logger({ collapsed: true })({ count: 0 });
```

## 옵션

- `collapsed`: (기본값: `false`) 로그 그룹을 기본적으로 축소합니다.
- `diff`: (기본값: `false`) 상태 변경의 깊은 비교(deep diff)를 표시합니다.
- `timestamp`: (기본값: `true`) 로그 헤더에 타임스탬프를 포함합니다.

## 운영 환경 동작

운영(production) 환경에서 `logger`는 자동으로 비활성화됩니다. 이는 불필요한 성능 저하를 방지하고 콘솔을 통한 민감한 정보의 노출을 막기 위함입니다. 로깅 로직 없이 원래의 스토어를 그대로 반환합니다.
