# @ilokesto/utilinent

`@ilokesto/utilinent`는 React에서 반복적으로 등장하는 렌더링 패턴을 더 선언적으로 표현하기 위한 유틸리티 모음입니다. 조건부 렌더링, 컬렉션 렌더링, viewport 기반 렌더링, 컴포지션, 확장성 도구를 하나의 패키지로 제공합니다.

## 설치

```bash
pnpm add @ilokesto/utilinent
```
