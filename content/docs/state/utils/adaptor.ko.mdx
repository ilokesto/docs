# `adaptor`

`adaptor`는 Immer의 `produce()`를 얇게 감싼 헬퍼입니다. draft 스타일의 객체 업데이트를 작성하면, 그 결과를 순수 updater 함수로 돌려줍니다.

```ts
import { adaptor } from '@ilokesto/state/utils';

type State = { user: { profile: { name: string } } };

const renameProfile = adaptor<State>((draft) => {
  draft.user.profile.name = 'New Name';
});

useStore.writeOnly()(renameProfile);
```

## 제약 사항

- `adaptor`는 객체 상태 형태에만 사용할 수 있습니다.
- 선택적 peer dependency인 `immer`에 의존합니다.

## 설치

```package-install
immer
```
