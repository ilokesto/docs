# @ilokesto/state

`@ilokesto/state`는 React를 위한 집중적인 상태 관리 라이브러리입니다. **스토어-훅의 통합**에 집중하여 단순함과 확장성 사이의 완벽한 균형을 제공하는 것을 목표로 합니다.

## 핵심 철학

- **Provider 없음**: 애플리케이션을 복잡한 Context Provider로 감쌀 필요가 없습니다.
- **예측 가능성**: 상태는 외부에 존재하지만 `useSyncExternalStore`를 통해 React의 렌더링과 깊게 통합됩니다.
- **조합 가능성**: 로깅, 영속화, 검증과 같은 미들웨어를 기본적으로 지원합니다.
- **성능**: 강력한 셀렉터 시스템을 통해 최적화된 리렌더링을 제공합니다.

## 설치

```package-install
@ilokesto/state
```

## 훑어보기

```tsx
import { create } from '@ilokesto/state';

const useCounter = create({ count: 0 });

function Counter() {
  const [count, setState] = useCounter((state) => state.count);

  return (
    <button onClick={() => setState((prev) => ({ ...prev, count: prev.count + 1 }))}>
      {count}
    </button>
  );
}
```
