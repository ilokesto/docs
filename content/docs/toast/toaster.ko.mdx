# Toaster (토스터)

`<Toaster />` 컴포넌트는 토스트가 렌더링되는 메인 영역입니다. 런타임을 등록하고 `toast()` 호출을 수신합니다.

## Props

### `toasterId` (string, 선택 사항)
기본값은 `"default"`입니다. `toast()` 호출을 이 특정 렌더러에 연결하는 데 사용합니다.

### `position` (`ToastPosition`, 선택 사항)
기본값은 `"top-right"`입니다. 가능한 값:
- `top-left`, `top-center`, `top-right`
- `bottom-left`, `bottom-center`, `bottom-right`

### `limit` (number, 선택 사항)
기본값은 `20`입니다. 한 번에 표시할 수 있는 최대 토스트 수입니다.

### `reverseOrder` (boolean, 선택 사항)
기본값은 `false`입니다. `true`로 설정하면 새로운 토스트가 스택 아래쪽에 나타납니다.

### `gutter` (number, 선택 사항)
기본값은 `8`입니다. 토스트 간의 간격(픽셀)입니다.

### `containerStyle` (React.CSSProperties, 선택 사항)
모든 토스트를 포함하는 컨테이너의 스타일입니다.

### `containerClassName` (string, 선택 사항)
토스트 컨테이너의 CSS 클래스입니다.

### `toastOptions` (`DefaultToastOptions`, 선택 사항)
이 렌더러에서 표시하는 모든 토스트에 적용되는 전역 옵션입니다. 여기서 지속 시간, 테마, 스타일을 전역적으로 설정할 수 있습니다.

### `transport` (`ToastTransport`, 선택 사항)
기본값은 `"inline"`입니다.
- `"inline"`: 일반 React 트리에서 토스트를 렌더링합니다.
- `"top-layer"`: 모든 다른 요소보다 위에 표시되도록 `popover` API를 사용하여 토스트를 렌더링합니다.

## 호버 시 일시 정지 (Hover Pause)

기본적으로 사용자가 `Toaster` 컨테이너에 마우스를 올리면 토스트의 사라짐 타이머가 일시 정지됩니다. 이를 통해 사용자는 긴 메시지가 너무 빨리 사라지지 않도록 읽을 수 있습니다.
