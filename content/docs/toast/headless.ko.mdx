# 헤드리스 사용

기본 row 렌더링 없이 toast 상태와 handler만 필요할 때 headless API를 쓰면 됩니다.

## `useToaster()`

```ts
const { toasts, handlers } = useToaster();
```

이 hook은 mounted `Toaster` context 아래에서만 동작합니다.

### `toasts`

해당 runtime의 현재 visible toast 목록입니다.

### `handlers`

- `updateHeight(id, height)`
- `startPause()`
- `endPause()`
- `calculateOffset(toast, options?)`

이 handler들은 기본 `Toaster`가 내부적으로 쓰는 것과 같은 도구입니다.

## 예제

```tsx
import { toast, useToaster } from '@ilokesto/toast';

function CustomToastList() {
  const { toasts, handlers } = useToaster();

  return (
    <div onMouseEnter={handlers.startPause} onMouseLeave={handlers.endPause}>
      {toasts.map((item) => (
        <div
          key={item.id}
          ref={(node) => {
            if (node) handlers.updateHeight(item.id, node.offsetHeight);
          }}
        >
          {item.message}
          <button onClick={() => toast.dismiss(item.id)}>Dismiss</button>
        </div>
      ))}
    </div>
  );
}
```

## `useToastItems(runtime)`

runtime을 이미 갖고 있고 visible item 목록만 원할 때는 이 lower-level hook을 사용하면 됩니다.
