# 검증 (Validate)

`validate` 미들웨어는 상태 업데이트를 반영하기 전에 Standard Schema v1 유효성 검사기를 사용하여 다음 상태를 확인합니다.

```ts
import { validate } from '@ilokesto/state/middleware';
import { z } from 'zod'; // Zod는 Standard Schema v1을 지원합니다

const schema = z.object({ count: z.number().min(0) });
const store = validate(schema)({ count: 0 });
```

## 주요 기능

- **Standard Schema v1**: Standard Schema v1을 지원하는 모든 스키마 라이브러리(Zod, Valibot, ArkType 등)와 호환됩니다.
- **차단형 검증**: 다음 상태가 유효하지 않으면 오류를 로그로 남기고 업데이트를 중단합니다.

## 제약 사항

- **비동기 지원**: 현재 구현에서는 비동기 스키마 유효성 검사를 지원하지 않습니다. 동기 유효성 검사만 가능합니다.
- **블로킹**: 유효성 검사는 업데이트가 반영되기 전에 발생하므로, 복잡한 유효성 검사 로직은 UI 반응성에 영향을 줄 수 있습니다.
