# API Reference

`@ilokesto/modal`은 named export만 제공합니다.

## Values

### `ModalProvider`

modal adapter를 `@ilokesto/overlay`에 연결해 실제 modal 렌더링을 호스팅합니다.

```tsx
<ModalProvider>{children}</ModalProvider>
```

선택적 prop:

- `store?` → custom overlay store, 전달하지 않으면 기본 global modal store 사용

### `useModal()`

다음 명령형 surface를 반환하는 React API입니다.

```ts
{
  open(options)
  display<T>(options)
  close(id, result?)
  remove(id?)
  clear()
}
```

중요한 동작:

- `open()`은 `id`를 반환합니다
- `display()`는 promise를 반환합니다
- `close()`는 closing phase를 시작합니다
- `remove()`는 즉시 제거합니다

### `modal`

같은 명령 surface를 제공하는 module-level facade입니다.

```ts
modal.open(options)
modal.display<T>(options)
modal.close(id, result?)
modal.remove(id?)
modal.clear()
```

### `globalModalStore`

facade와 `ModalProvider`의 기본값에서 사용하는 overlay store입니다.

## Types

### `ModalProps`

지원하는 필드:

- `children?: ReactNode`
- `transport?: 'inline' | 'top-layer'`
- `position?: ModalPosition`
- `dismissible?: boolean`
- `onDismiss?: () => void`
- `className?: string`
- `style?: React.CSSProperties`
- `backdropClassName?: string`
- `backdropStyle?: React.CSSProperties`
- `autoFocus?: boolean`
- `restoreFocus?: boolean`

### `ModalPosition`

```ts
'center'
| 'top'
| 'bottom'
| 'left'
| 'right'
| 'top-left'
| 'top-right'
| 'bottom-left'
| 'bottom-right'
```

### 그 외 export되는 타입

- `ModalProviderProps`
- `UseModalOptions`
- `ModalFacadeOptions`
- `ModalAdapterProps`
